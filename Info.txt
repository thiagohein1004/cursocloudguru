Comandos git:

git clone repo = clona o repositorio
git branch -d novabranch = cria e vai pra nova branch
git checkout branch = altera de branch
git log = verifica os logs
git status = verifica o status
git add = adiciona os objetos
git commit -m "mensagem" = faz o commit
git push -u origin novabranch = manda a nova branch pro github
git merge novabranch = faz o merge pra main (precisa estar na branch main)
git push = faz o push pra merge
